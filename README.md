# Hashcat WiFi Cracker - ฉบับไทย

เครื่องมือแคร็กรหัส WiFi อัตโนมัติที่ปรับแต่งสำหรับเน็ตไทยโดยเฉพาะ ใช้ Hashcat ในการทำงาน

> 📖 **Read in English:** [Click here](en/README.md)

## ความสามารถ

- ✅ ปรับแต่งสำหรับเน็ต WiFi ไทย - เน้นรูปแบบรหัสที่คนไทยนิยมใช้
- ⏱️ จับเวลาการทำงาน - แสดงระยะเวลาที่ใช้ในการแคร็ก
- 🎯 ลำดับการโจมตีอัจฉริยะ - เริ่มจากวิธีที่เร็วที่สุดก่อน
- 🔍 ตรวจสอบซ้ำ - ข้ามรหัสที่แคร็กไปแล้วอัตโนมัติ
- 🇹🇭 รูปแบบเบอร์มือถือไทย - ครอบคลุมทุกเครือข่าย (061-069, 080-099)

## สิ่งที่ต้องมี

- **Hashcat** (v6.0.0 ขึ้นไป) - [ดาวน์โหลด](https://hashcat.net/hashcat/)
- **Windows** (สคริปต์รองรับ Windows)
- **GPU** (แนะนำ NVIDIA/AMD สำหรับความเร็ว)
- **rockyou.txt** - รายการรหัสผ่าน [ดาวน์โหลด](https://github.com/brannondorsey/naive-hashcat/releases)
- **ไฟล์ .hc22000** - ไฟล์ handshake ที่จับได้จาก WiFi

## วิธีใช้งาน

### แบบง่าย: ลากไฟล์วาง
1. ลากไฟล์ `.hc22000` ของคุณมาวางบน `CRACK_V.bat`
2. รอจนกว่าจะแคร็กเสร็จ

### แบบ Command Line
```cmd
cd C:\hashcat-7.1.2
CRACK_V2.bat your_capture.hc22000
```

## แก้ปัญหา

### "hashcat.exe not found"
- ตรวจสอบว่าสคริปต์อยู่โฟลเดอร์เดียวกับ `hashcat.exe`

### "rockyou.txt not found"
- ดาวน์โหลดจาก: https://github.com/brannondorsey/naive-hashcat/releases
- วางไว้โฟลเดอร์เดียวกับสคริปต์

### "No hashes loaded"
- ไฟล์ `.hc22000` อาจเสียหาย ลองจับ handshake ใหม่

### ช้ามาก
- ตรวจสอบว่า GPU ทำงานหรือไม่ (ดูที่ "Device #1" ในผลลัพธ์)
- อัพเดทไดรเวอร์การ์ดจอ
- ปิดโปรแกรมอื่นที่ใช้ GPU

## เบอร์มือถือไทยที่รองรับ

**AIS:** 061-069, 080-083, 084-089  
**DTAC:** 090-099  
**TrueMove H:** 061-069, 092-094

รวม: 29 prefix × 10 ล้านเบอร์ = 290 ล้านรูปแบบ

## รูปแบบวันเกิด

ทดสอบรูปแบบวันเกิดที่นิยม:
- **พ.ศ.:** 2500-2599 (ค.ศ. 1957-2056)
- **ค.ศ.:** 1900-1999, 2000-2099
- **รูปแบบ:** DDMMYYYY (เช่น 01012540, 25121990)

## คำเตือนด้านกฎหมาย

⚠️ **สำคัญ:** เครื่องมือนี้ใช้เพื่อการศึกษาและทดสอบที่ได้รับอนุญาตเท่านั้น

**ใช้ได้ถูกกฎหมาย:**
- ทดสอบเน็ต WiFi ของตัวเอง
- ทดสอบความปลอดภัยที่ได้รับอนุญาตเป็นลายลักษณ์อักษร
- การศึกษาวิจัยด้านความปลอดภัย

**ผิดกฎหมาย:**
- แคร็กรหัส WiFi โดยไม่ได้รับอนุญาต
- เข้าถึงเครือข่ายที่ไม่ใช่ของคุณ

**ข้อจำกัดความรับผิด:** ผู้พัฒนาไม่รับผิดชอบต่อการใช้งานที่ผิดกฎหมาย ขอให้ได้รับอนุญาตก่อนทดสอบเครือข่ายใดๆ

## คำถามที่พบบ่อย

**ถาม: ใช้เวลานานแค่ไหน?**  
ตอบ: ขึ้นอยู่กับความซับซ้อนของรหัสและ GPU รหัสง่ายๆ: ไม่กี่นาที, เลข 8 หลัก: 3-30 นาที, เลข 9 หลัก: 30-120 นาที

**ถาม: ใช้กับ WPA3 ได้ไหม?**  
ตอบ: ไม่ได้ รองรับเฉพาะ WPA/WPA2 เท่านั้น

**ถาม: หยุดแล้วทำต่อได้ไหม?**  
ตอบ: ได้! กด `Ctrl+C` เพื่อหยุด แล้วรันใหม่เพื่อทำต่อ รหัสที่แคร็กแล้วจะถูกบันทึกใน `hashcat.potfile`

**ถาม: GPU ไม่ทำงานทำไง?**  
ตอบ: ตรวจสอบว่าติดตั้งไดรเวอร์ GPU แล้ว ดูที่ผลลัพธ์ต้องมี "Device #1" แสดงชื่อการ์ดจอ
